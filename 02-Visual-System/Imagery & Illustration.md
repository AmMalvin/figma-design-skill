# Imagery & Illustration

## Purpose

This module teaches the AI how to build a consistent visual language using photography, illustrations, graphics, avatars, and supporting visual assets.

Every image should communicate purpose.

Visual assets should strengthen usability before aesthetics.

---

# Definition

Imagery and illustration define the visual storytelling system of a product.

They communicate information, reinforce branding, support recognition, and improve emotional engagement.

Every visual asset should belong to one unified system.

---

# Objectives

After completing this module, the AI should be able to:

- Build consistent imagery systems
- Create illustration guidelines
- Improve visual storytelling
- Strengthen branding
- Improve accessibility
- Standardize image usage
- Create reusable asset libraries
- Improve developer implementation

---

# Visual Mindset

Never ask:

"Does this image look beautiful?"

Always ask:

- Does it communicate clearly?
- Does it support the user task?
- Is it consistent with the design system?
- Is it accessible?
- Can it be reused?

---

# Core Principles

Always:

- Prioritize communication.
- Use authentic visuals.
- Maintain consistency.
- Support accessibility.
- Optimize performance.
- Build reusable assets.

Never:

- Use decorative images without purpose.
- Mix illustration styles.
- Distort images.
- Ignore accessibility.
- Use inconsistent visual quality.

---

# Why Visual Assets Exist

Visual assets should:

- Explain
- Guide
- Educate
- Reinforce branding
- Build trust
- Improve recognition
- Support onboarding
- Improve emotional connection

Visuals should never distract from primary tasks.

---

# Visual Asset Categories

Every product should define:

- Photography
- Illustration
- Graphics
- Avatars
- Empty States
- Onboarding Artwork
- Marketing Assets
- Product Images
- Background Graphics
- Decorative Assets

Each category should have documented rules.

---

# Visual Consistency

Every visual asset should maintain consistency in:

- Style
- Perspective
- Color treatment
- Lighting
- Composition
- Quality
- Resolution
- Cropping

Consistency improves product recognition.

---

# Brand Expression

Imagery should reinforce:

- Brand personality
- Product values
- User expectations
- Emotional tone

Visual identity should remain recognizable across every screen.

---

# AI Decision Rules

Before approving any visual asset, answer:

- Does it communicate clearly?
- Does it belong to the visual system?
- Is accessibility supported?
- Is quality sufficient?
- Can it be reused?
- Does it strengthen the brand?

If any answer is negative, redesign the asset.

---

# Validation Checklist

□ Visual language defined

□ Asset categories documented

□ Brand consistency reviewed

□ Accessibility reviewed

□ Reusability confirmed

□ Documentation updated
