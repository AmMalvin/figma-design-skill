# Cognitive Psychology

## Purpose

This module teaches the AI how humans perceive, process, understand, remember, and respond to digital interfaces.

Every design decision should align with how the human brain naturally works.

The objective is to reduce cognitive effort while increasing comprehension, confidence, speed, and accuracy.

Interfaces should adapt to human cognition.

Humans should never be forced to adapt to interfaces.

---

# Definition

Cognitive Psychology studies the mental processes responsible for perception, attention, memory, learning, reasoning, language, and decision making.

Within digital product design, these principles explain why users become confused, make mistakes, abandon tasks, or successfully complete them.

---

# Objectives

After completing this module, the AI should be able to:

- Reduce cognitive load
- Improve comprehension
- Improve decision making
- Reduce user errors
- Improve memory support
- Improve learnability
- Improve discoverability
- Improve navigation
- Improve confidence
- Improve task completion

---

# Cognitive Design Mindset

Never ask:

"Does this interface look modern?"

Always ask:

- Can users understand it immediately?
- Does it reduce thinking?
- Does it reduce memory requirements?
- Does it match human expectations?
- Does it support attention?
- Does it guide decision making?
- Does it build confidence?

---

# Human Information Processing

Every interaction follows this sequence:

Stimulus

↓

Attention

↓

Perception

↓

Interpretation

↓

Memory

↓

Decision

↓

Action

↓

Feedback

↓

Learning

Every stage should be optimized.

---

# Cognitive Design Principles

Always:

- Reduce unnecessary thinking.
- Design for recognition.
- Support memory.
- Guide attention.
- Build predictable experiences.
- Reduce uncertainty.
- Maintain consistency.
- Simplify decisions.
- Provide immediate feedback.

Never:

- Overload working memory.
- Require memorization.
- Introduce unnecessary complexity.
- Hide important information.
- Surprise users without reason.
- Increase cognitive effort.

---

# The Brain's Primary Goals

When interacting with software, people naturally seek to:

- Understand quickly
- Complete tasks efficiently
- Avoid mistakes
- Minimize effort
- Feel confident
- Maintain control
- Build trust

Every interface should support these goals.

---

# AI Decision Rules

Before recommending any interface:

- Does this reduce mental effort?
- Does this improve understanding?
- Does this simplify decisions?
- Does this improve learning?
- Does this reduce memory requirements?
- Does this reduce mistakes?
- Does this improve confidence?

If any answer is negative, revise the solution.
